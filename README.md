# hello-kafka

Project is created with:
* Micronaut;
* Kotlin;
* Kafka;
* GRPC;
* BloomRPC;

## Setup
To run this project:

Use Java11 with:
* sdkman: ``sdk install java 11.0.11.hs-adpt``;
* sdkman: ``sdk use java 11.0.11.hs-adpt``;

Build and run project with:
* Execute: ``docker-compose up -d``;
* Execute: `` ./gradlew clean build``;
* Execute: `` ./gradlew run``;
